# git-bookiza

Bookiza portable git commands

```
  npm i -S git-bookiza

  const git = require('git-bookiza');
```

Three commands that are available:

```

  const git = require('git-bookiza');
  
  - git.init()    => $ bookiza new
  - git.push()    => $ bookiza push
  - git.status()  => $ bookiza status
